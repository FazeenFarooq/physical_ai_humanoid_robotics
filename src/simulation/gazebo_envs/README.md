# Gazebo Environment Configuration for Physical AI Course

This directory contains configuration files and world definitions for Gazebo Garden simulation environments used in the Physical AI & Humanoid Robotics course.

## World Files
- `basic_room.world`: Simple indoor environment with walls and basic furniture
- `lab_space.world`: Laboratory environment with various obstacles and test objects
- `outdoor_area.world`: Outdoor environment for navigation testing

## Robot Models
- `humanoid_model.urdf`: Basic humanoid robot model for simulation
- `sensor_config.xacro`: Sensor configuration for the robot (RGB-D, LiDAR, IMU)

## Plugins
- `custom_sensors.so`: Custom sensor plugins for physics-accurate simulation
- `robot_control.so`: Basic robot control interface for simulation
